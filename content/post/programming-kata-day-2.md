Problem: Write a function to calculate fibonacci number in constant space complexity.

{{< highlight clojure >}}

(defn fib [n]
  (loop [a 0
         b 1
         i n]
    (if (= 0 i)
      a
      (recur b (+ a b) (dec i)))))
